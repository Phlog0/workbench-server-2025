# Workbench Server ⚡

**Backend для конструктора электрических схем** — мощный API, управляющий данными проектов, пользователями и типовыми решениями для электроэнергетики.

Сервер обеспечивает безопасное хранение схем, работу с Excel-файлами и систему авторизации.

## Стек технологий

- **Фреймворк:** NestJS
- **База данных:** PostgreSQL + Prisma
- **Аутентификация:** JWT (Access + Refresh токены)
- **Файлы:** Обработка Excel (XLSX, XLS), работа с изображениями
- **Документация:** Swagger (OpenAPI)

## Ключевые возможности

### 🔐 Безопасность и авторизация

- **JWT-аутентификация** с разделением на Access и Refresh токены
- **Хеширование паролей** (bcrypt)

### 📁 Управление проектами

- CRUD для электрических схем
- Сохранение метаданных (автор, дата создания, техническая документация проекта)
- Хранение JSON-схем в структурированном виде

### 👥 Пользователи и роли

- Регистрация / Авторизация

# Как воспользоваться:

1. Скачать репозиторий
2. npm install
3. npx prisma migrate deploy
4. npx prisma generate
5. npm run build
